# 🎵 Mabinogi MML Converter

AI 기반 음악 파일을 마비노기 MML로 자동 변환하는 웹 도구

**Made by 란님 | 알리사**  
**Powered by Antigravity**

---

## ✨ 주요 기능

- 🎹 **MIDI → MML**: MIDI 파일을 즉시 변환
- 🎤 **MP3/WAV → MML**: AI가 오디오를 악보로 변환
- ⚡ **구간 선택**: 원하는 부분만 빠르게 변환
- 🎚️ **세밀한 조정**:
  - BPM 조절
  - 박자 보정 (1/32, 1/64, 1/128)
  - 트랙별 음량 조절
  - 노이즈 필터링

---

## 🚀 사용 방법

1. **파일 업로드**: MIDI 또는 MP3/WAV 파일 드래그 앤 드롭
2. **구간 설정** (선택): 변환할 시작/종료 시간 입력
3. **분석 시작**: AI가 자동으로 음표 인식
4. **설정 조정**:
   - BPM 확인/수정
   - 음량 조절 (0-15)
   - 필터 설정 (최소 길이, 최소 강도)
5. **변환하기**: MML 생성
6. **전체 복사**: 마비노기에 바로 붙여넣기

---

## 🎯 최적화 팁

### 오디오 파일 변환 시:
- ✅ **깨끗한 음원** 사용 (반주 없는 MR 추천)
- ✅ **BPM 정확히 입력** (박자 오차 최소화)
- ✅ **구간 선택** 활용 (속도 3배 향상)
- ✅ **필터 조절**:
  - 잡음 많음 → 최소 길이 ↑ (100ms)
  - 유령음 많음 → 최소 강도 ↑ (20%)

### MIDI 파일 변환 시:
- ✅ 트랙이 3개 이하인 MIDI 권장
- ✅ 복잡한 화음은 자동으로 3트랙으로 분배
